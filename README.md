# Тестовое задание QA Авито - Задание 2.1

## Описание проекта

Автоматизированные API тесты для микросервиса объявлений Авито.

**Host**: https://qa-internship.avito.com

## Структура

- `TESTCASES.md` - Ручные тест-кейсы для API
- `tests.py` - Автоматизированные тесты на pytest
- `BUGS.md` - Найденные баги в API

## Установка и запуск

### Требования

- Python 3.13+
- uv (пакетный менеджер)

### Установка зависимостей

```bash
uv sync
```

### Запуск тестов

```bash
# Запуск всех тестов
uv run pytest tests.py -v

# Запуск с подробным выводом
uv run pytest tests.py -v -s
```
