# Баг-репорты API микросервиса Авито

## Найденные баги

### BUG-001: Отсутствует валидация отрицательных цен при создании объявлений
**Серьезность:** Рекомендация к улучшению

**Описание проблемы:**
API позволяет создавать объявления с отрицательными значениями цены, что недопустимо для бизнес-логики. Отрицательные цены могут привести к ошибкам в расчетах стоимости объявлений и проблемам с отчетностью.

**curl для создания объявления:**
```bash
curl --location 'https://qa-internship.avito.com/api/1/item' \
--header 'Content-Type: application/json' \
--header 'Accept: application/json' \
--data '{
  "sellerID": 123,
  "name": "testItem",
  "price": -9900,
  "statistics": {
    "likes": 21,
    "viewCount": 11,
    "contacts": 43
  }
}'
```
